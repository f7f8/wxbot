# wxbot
> 微信智能服务助手。

## 特点
1. 接口最大程度采用 [REST](https://zh.wikipedia.org/wiki/REST) 风格，简单易用。
2. 接口交换数据统一使用 [json](https://zh.wikipedia.org/wiki/JSON) 格式。

## 上行接口

### 获取待处理队列
> GET /wxbot/queue

路径参数：
>
（无）

请求参数：
>
（无）

返回结果：
>
```
// 任务队列
{
    "get_queue_response": [
        "31415926",
        "31415927",
        "31415928",
        "31415929"
    ]
}
```

### 上传微信登录二维码
> POST /wxbot/{***owner***}/qrcode

路径参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| owner         | string    | 群主ID            | 是    | 31415926                                      |

请求参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| code          | string    | 二维码            | 是    | https://login.weixin.qq.com/l/4d-0BmfrOg==    |

返回结果：
>
```
// 上传成功
{
    "qrcode_response": {
        "result": "ok"
    }
}
```

### 获取群信息
> GET /wxbot/{***owner***}/info

路径参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| owner         | string    | 群主ID            | 是    | 31415926                                      |

请求参数：
>
（无）

返回结果：
>
```
// 群基本信息
{
    "info_response": {
        "owner": "31415926",
        "name": "404枫人院",
        "expire": "2016-07-31"
    }
}
```

### 更新群信息
> PUT /wxbot/{***owner***}/info

路径参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| owner         | string    | 群主ID            | 是    | 31415926                                      |

请求参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| name          | string    | 群名称            | 是    | 404枫人院(2)号群                              |

返回结果：
>
```
// 更新群信息成功
{
    "update_info_response": {
        "result": "ok"
    }
}
```

### 请求入群
> POST /wxbot/{***owner***}/member

路径参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                          |
| :------------ | :-------- | :---------------- | :---- | :-------------------------------------------- |
| owner         | string    | 群主ID            | 是    | 31415926                                      |

请求参数：
>
| 参数          | 类型      | 说明              | 必填  | 示例                                      |
| :------------ | :-------- | :---------------- | :---- | :---------------------------------------- |
| token         | string    | 入群令牌          | 是    | NTkxNjkwNzE4RDY2                          |
| nickname      | string    | 群员昵称          | 是    | 黑旋风                                    |

返回结果：
>
```
// 入群成功
{
    "membership_join_response": {
        "owner": "31415926",
        "name": "404枫人院"
    }
}
```
